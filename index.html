How the Web Works

In your own terms, define the following terms:

  • What is HTTP

    • Hypertext Transfer Protocal, or how the servers and browsers communicate

  • What is a URL?

    • an address for some resourse

  • What is DNS?

    • Converts URLs into IP Addresses

  • What is a query string?

    • allows you to pass key-value pairs into the URL

  • What are two HTTP verbs and how are they different?

    • Get: gets data from servers / Post: Sends some data to servers

  • What is an HTTP request?

    • A request from a user to the server

  • What is an HTTP response?

    • the servers response to a users request

  • What is an HTTP header? Give a couple examples of request and response headers you have seen.

    • Provides additional info about the HTTP request, some examples are: last modified, host, cookies, content type

  • What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser?

    • The browser turns the url into an IP address, then it makes a request to that IP address server, the server gives a response, the browser then generates a DOM from the server response
